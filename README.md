# Hanbitco API Python

Official API Python Wrapper for [Hanbitco Exchange](https://www.hanbitco.com).

## Get Started

### Run with source code

Install dependencies

```bash
pipenv install
```

### Download from PyPi

```bash
pip3 install hanbitco-api
```
